## Project Overview
A full-stack solution for MSME financing implementing lender assignment logic using:
- Credit score analysis
- Document verification
- SQLite for local data storage
- Sequelize ORM for database management

## Key Features
**MSME Portal**
- Application submission with document tracking
- Real-time credit limit simulation
- Local database storage for rapid prototyping

**Lender Portal**
- SQLite-based program configuration
- Rule management with DBeaver compatibility
- Offline-first capability

## Tech Stack
**Frontend**
- React 18 with Vite
- Tailwind CSS 3
- Axios for API communication

**Backend**
- Node.js 18
- Express.js 4
- SQLite 3 (via Sequelize ORM)
- DBeaver for database management

**Development**
- File-based database storage
- Cross-platform DB compatibility
- Lightweight local development setup
